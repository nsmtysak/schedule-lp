The old way

その予定共有、
まだ手で書いていますか？

ホワイトボードへの手書きは、書ける人・見られる人・タイミングが限られます。外からは更新できず、消し忘れや読み間違いも。「あの人どこ？」が毎日くり返されます。

  • 外出先からは書けない／事務所にいる人しか最新を把握できない
  • 消し忘れ・読み間違い／手書き文字の解釈ちがいで連絡ミス
  • 実績はまた別でExcel入力／二度手間で、記録も抜けやすい
Solution

入力するだけで、自動で振り分け。

アプリに書き込むだけで、予定はGoogleカレンダーへ、実績はスプレッドシートへ自動で振り分け。Google Apps Script で動くため、サーバーもデータベースも不要です。

入力フォーム

名前を選んで入力。PC・スマホ共通のWeb画面。

自動振り分け

会社の1アカウントが裏側で処理。ログイン不要。

カレンダー＆シート

予定はカレンダー、実績は表に自動で蓄積。

追加費用 ¥0
Googleカレンダー・スプレッドシート・Apps Script はいずれも無料。5人規模なら無料枠で十分に収まり、月額のランニングコストをかけずに始められます。利用者はGoogleアカウント不要。
Benefits

手書き・手入力を、仕組みでなくす。

移動中でも記録できる体制へ。入れ忘れも転記ミスも、人の注意ではなく仕組みで防ぎます。

その場で入力できる

外出先のスマホから予定登録、現場で実績入力。事務所に戻る前に作業が終わります。

全員の動きが一目で

担当者ごとに色分け。誰がどこにいるかが即わかり、掲示板の読み間違いがなくなります。

入れ忘れを防ぐ

実績が未入力の予定はカードが色づく仕組み。「記録の抜け」にその場で気づけます。

転記ミスがなくなる

予定を実績にそのまま引き継ぎ、所要時間も時刻から自動計算。二重入力や打ち間違いを防ぎます。

コストをかけない

既存のGoogleサービスを活用。月額料金や専用ソフトの購入が不要で、低リスクで始められます。

あとから直せる

登録した予定・実績は編集・削除が可能。担当者の追加・削除も設定画面から行えます。

5つのタブで、迷わず完結。

  • 予定 / 実績

    出発前に予定を登録、帰社前に実績を記録。今日の予定をタップすれば内容を引き継ぎます。

  • 一覧 / メンバー

    自分や全員の予定・実績を日付ごとに確認。未入力の予定はひと目で分かります。

  • 設定

    名前を選ぶだけで利用開始。担当者カラーの割り当てや追加・削除も自由。

On site

現場でも、外回りでも。

出発前のスマホ登録から、移動中の確認、帰社前の実績入力まで。日々の流れにそのまま溶け込みます。

How to Use

2つの場面を、タブ切り替えで。

出発前

予定を登録する

  1. 最初に一度だけ自分の名前を選ぶ（次回から自動）
  2. 行き先・目的・日付・開始/終了時刻を入力
  3. 登録で、担当者カラーの共有カレンダーに反映
  4. ホワイトボードの代わりに、全員がスマホで確認
帰社後

実績を入力する

  1. 実績タブで今日の予定をタップして引き継ぐ
  2. 開始・終了時刻を選ぶと所要時間を自動計算
  3. 記録でスプレッドシートに自動追記
  4. 予定のない飛び込み作業もゼロから入力可能
FAQ

よくあるご質問

利用者全員にGoogleアカウントが必要ですか？
不要です。アプリは会社の1アカウントの権限で動き、利用者は最初に名前を選ぶだけ。ログインもパスワードもいりません。全員が同じURLを開いて使えます。
月額費用やサーバー契約は？
Googleカレンダー・スプレッドシート・Apps Script はいずれも無料で、5人規模なら無料枠で十分です。サーバーやデータベースの契約も不要。月額のランニングコストをかけずに始められます。
スマホアプリのインストールは必要ですか？
不要です。配布されたURLを開き、ホーム画面に追加すれば、通常のアプリのように1タップで起動できます。PCも同じURLで使えます。
データはどこに保存され、誰のものになりますか？
予定は御社のGoogleカレンダー、実績は御社のGoogleスプレッドシートに蓄積されます。データは御社のGoogleアカウント内にあり、いつでも閲覧・書き出しが可能です。
担当者は5人までですか？
5人規模を想定した設計ですが、担当者の追加・削除は設定画面から行えます。人数や運用に合わせた調整はご相談ください。
登録した予定・実績は後から直せますか？
はい。登録後の予定・実績は編集・削除が可能です。打ち間違いや予定変更にもその場で対応できます。

そのホワイトボード、
今日で卒業しませんか。

サーバー契約も初期費用も不要。共有カレンダーの開設から着手すれば、最短で「ホワイトボードに手書き」の運用を卒業できます。御社の運用に合わせて設定してご案内します。
