こんな現場、ありませんか

その「手作業」が、
ミスと手間を生む。

予定共有も実績記録も“手作業”が前提のため、転記ミス・確認の手間・属人化が積み重なっています。

  • 掲示板にマジックで手書き

    外出先からは書けず、消し忘れ・読み間違いも。事務所にいる人しか最新状況を把握できません。

  • 帰社後にExcelへ手入力

    誰が・どこで・何を・所要時間を毎回手打ち。入力が後回しになり、記録が抜ける・ばらつくことも。

  • 予定と実績がバラバラ

    掲示板とExcelで保管場所が分かれ、見比べるのに手間。全員の動きを一目で確認できません。

  • 事務所に戻らないと作業できない

    記入も入力も会社のPC・掲示板の前でしか行えず、移動や直行直帰のロスにつながります。

Solution

入力するだけで、
自動で振り分け。

アプリに書き込むだけで、予定はGoogleカレンダーへ、実績はスプレッドシートへ自動で振り分け。Google Apps Script で動くため、サーバーもデータベースも不要です。

入力フォーム

名前を選んで内容を入力。PC・スマホ共通のWeb画面。

自動振り分け

会社の1アカウントが裏側で処理。利用者はログイン不要。

カレンダー＆シート

予定はGoogleカレンダー、実績は表に自動で蓄積。

追加費用 ¥0
Googleカレンダー・スプレッドシート・Apps Script はいずれも無料。5人規模なら無料枠で十分に収まり、月額のランニングコストをかけずに運用を始められます。

※ 利用者はGoogleアカウント不要。会社で1つ用意すれば、全員が同じURLで使えます。

Main Screens

5つのタブで、迷わず完結。

画面下（PCは左サイド）のタブを切り替えるだけ。出発前・帰社後・確認のすべてが、ひとつのアプリにまとまっています。

予定

行き先・目的・日付・開始/終了時刻（10分刻み）を入れて登録。担当者カラーで共有カレンダーへ。

実績

今日の予定をタップすれば内容を引き継ぎ。開始・終了から所要時間を自動計算してシートへ記録。

一覧

自分のその日の予定・実績をまとめて確認。実績が未入力の予定はカードが色づき、入れ忘れを防止。

メンバー

自分以外のメンバーの予定・実績を日付ごとに閲覧。誰がどこにいるかがすぐ分かります。

設定

自分の名前を選択。担当者の追加・削除や、実績スプレッドシートへの移動もここから。

Benefits

手書き・手入力を、
仕組みでなくす。

移動中でも記録できる体制へ。入れ忘れも転記ミスも、人の注意ではなく仕組みで防ぎます。

コストをかけない

既存のGoogleサービスを活用するため、月額料金や専用ソフトの購入が不要。低リスクで始められます。

その場で入力できる

外出先のスマホから予定登録、現場で実績入力。事務所に戻る前に作業が終わります。

全員の動きが一目で

担当者ごとに色分け。メンバータブで誰がどこにいるかが即わかり、掲示板の読み間違いもゼロに。

入れ忘れを防ぐ

実績が未入力の予定はカードが色づいて目立つ仕組み。「記録の抜け」にその場で気づけます。

転記ミスがなくなる

予定の内容を実績にそのまま引き継ぎ。所要時間も時刻から自動計算し、二重入力や打ち間違いを防ぎます。

あとから直せる・増やせる

登録した予定・実績は編集や削除が可能。担当者の追加・削除も設定画面から自由に行えます。

How to Use

2つの場面を、タブ切り替えで。

スマホはホーム画面に追加すれば、アプリのように1タップで起動。操作はどちらも数ステップで完了します。

出発前

予定を登録する

  1. 最初に一度だけ自分の名前を選ぶ（次回から自動）
  2. 行き先・目的・日付・開始/終了時刻を入力
  3. 登録を押すと、担当者カラーで共有カレンダーに反映
  4. 掲示板の代わりに、全員がスマホで確認できる
帰社後

実績を入力する

  1. 実績タブで、今日の予定をタップして引き継ぐ
  2. 開始・終了時刻を選ぶと所要時間を自動計算
  3. 記録でスプレッドシートに自動追記
  4. 予定のない飛び込み作業も、ゼロから入力できる
FAQ

よくあるご質問

利用者全員にGoogleアカウントが必要ですか？
不要です。アプリは会社の1アカウントの権限で動き、利用者は最初に名前を選ぶだけ。ログインもパスワードもいりません。全員が同じURLを開いて使えます。
月額費用やサーバー契約は？
Googleカレンダー・スプレッドシート・Apps Script はいずれも無料で、5人規模なら無料枠で十分に収まります。サーバーやデータベースの契約も不要。月額のランニングコストをかけずに始められます。
スマホアプリのインストールは必要ですか？
不要です。配布されたURLを開き、ホーム画面に追加すれば、通常のアプリのように1タップで起動できます。PCも同じURLでそのまま使えます。
データはどこに保存され、誰のものになりますか？
予定は御社のGoogleカレンダー、実績は御社のGoogleスプレッドシートに蓄積されます。データは御社のGoogleアカウント内にあり、いつでも閲覧・書き出しが可能。外部サービスに預ける必要はありません。
担当者は5人までですか？
5人規模を想定した設計ですが、担当者の追加・削除は設定画面から行えます。人数や運用に合わせた調整についてはご相談ください。
登録した予定・実績は後から直せますか？
はい。登録後の予定・実績は編集・削除が可能です。打ち間違いや予定変更にもその場で対応できます。

まずは小さく、
すぐに始められます。

サーバー契約も初期費用も不要。共有カレンダーの開設から着手すれば、最短で「掲示板にマジック」の運用を卒業できます。御社の運用に合わせて設定してご案内します。
