On site

現場で、こう使う。

出発前のスマホ登録から、現場での確認、帰社前の実績入力まで。日々の流れにそのまま溶け込みます。

How to Use

2つの場面を、タブ切り替えで。

出発前

予定を登録する

  1. 最初に一度だけ自分の名前を選ぶ（次回から自動）
  2. 行き先・目的・日付・開始/終了時刻を入力
  3. 登録で、担当者カラーの共有カレンダーに反映
  4. 掲示板の代わりに、全員がスマホで確認
帰社後

実績を入力する

  1. 実績タブで今日の予定をタップして引き継ぐ
  2. 開始・終了時刻を選ぶと所要時間を自動計算
  3. 記録でスプレッドシートに自動追記
  4. 予定のない飛び込み作業もゼロから入力可能
FAQ

よくあるご質問

利用者全員にGoogleアカウントが必要ですか？
不要です。アプリは会社の1アカウントの権限で動き、利用者は最初に名前を選ぶだけ。ログインもパスワードもいりません。全員が同じURLを開いて使えます。
月額費用やサーバー契約は？
Googleカレンダー・スプレッドシート・Apps Script はいずれも無料で、5人規模なら無料枠で十分です。サーバーやデータベースの契約も不要。月額のランニングコストをかけずに始められます。
スマホアプリのインストールは必要ですか？
不要です。配布されたURLを開き、ホーム画面に追加すれば、通常のアプリのように1タップで起動できます。PCも同じURLで使えます。
データはどこに保存され、誰のものになりますか？
予定は御社のGoogleカレンダー、実績は御社のGoogleスプレッドシートに蓄積されます。データは御社のGoogleアカウント内にあり、いつでも閲覧・書き出しが可能です。
担当者は5人までですか？
5人規模を想定した設計ですが、担当者の追加・削除は設定画面から行えます。人数や運用に合わせた調整はご相談ください。
登録した予定・実績は後から直せますか？
はい。登録後の予定・実績は編集・削除が可能です。打ち間違いや予定変更にもその場で対応できます。
