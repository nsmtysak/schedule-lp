# 予定実績シェア — 拡販LP

GAS製の「予定・実績 共有システム」（schedule-app）を、**現場を持つ中小企業に売る**ための
ランディングページ（単体・1ページ・依存ゼロ）。

- 読み手: 現場チームを持つ中小企業（設備・建設・訪問業務など）
- ゴール: 「会社での導入を相談する」問い合わせ（mailto）
- 方向性: 実写を主役にした鮮やかな広告ビジュアル（青ブランド＋グリーンのアクセント）

## ファイル構成

```
schedule-lp/
├── index.html        # LP本体（CSS/JS同梱・1ファイル完結）
├── images/           # Pexelsの実写素材（商用利用可）
│   ├── hero.jpg      # ヒーロー：工場でスマホを確認する作業者
│   ├── problem.jpg   # 課題：デスクで紙の資料を扱う
│   ├── band.jpg      # 引用バンド：バンの運転席でスマホ（直行直帰）
│   ├── cta.jpg       # 最終CTA：スマホを見る作業者（シネマティック）
│   └── worker-phone.jpg # 予備カット
└── README.md
```

## 設置

`index.html` と `images/` をそのままサーバー／GitHub Pages 等に置くだけ。ビルド不要。

## 元アプリの内容（訴求の根拠）

- 5人規模の現場チーム向け。掲示板＋Excelを置き換え、**予定→Googleカレンダー／実績→スプレッドシート**へ自動振り分け。
- ログイン不要（会社の1アカウントで動作）、追加費用ゼロ（無料枠）、PC・スマホ共通。
- 画面: 予定／実績／一覧／メンバー／設定の5タブ。担当者カラー、所要時間の自動計算、未入力の可視化、編集・削除可。
- 技術: Google Apps Script ＋ Googleカレンダー／スプレッドシート連携。
- 元アプリ: `../schedule-app`（提案資料 `docs/proposal.html` を訴求の素材として活用）。

## デザイン方針

- ブランド: 青(#2563eb)＝信頼／グリーン(#13b46a)＝効率・CTA・完了／ネイビー(#0c1322)＝引き締め。
- 書体は Noto Sans JP に統一。記号・絵文字は使わずインラインSVGアイコンで統一。
- 見出しは文節を `white-space:nowrap` で束ね、スマホで語中改行しない（PC専用改行は `br.pc`）。
- 構成: ヒーロー → 数字バンド → 課題(写真分割) → 解決(フロー・ダーク) → 画面(5タブ＋スマホmock) → 引用バンド(全面写真) → メリット6 → 使い方 → FAQ → 最終CTA(写真) → フッター。
